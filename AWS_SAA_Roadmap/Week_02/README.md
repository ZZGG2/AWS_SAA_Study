### 1. VPC (Virtual Private Cloud)
- **CIDR**: Classless Inter-Domain Routing.
- **Subnets**: Public vs Private.

### 2. Routing & Gateways
- **Route Table**: Rules for where network traffic is directed.
- **Internet Gateway (IGW)**: Enables access to the internet.
- **NAT Gateway**: Allows private subnets to access the internet.

### 3. Security
- **Security Group**: Stateful firewall at the instance level.
- **NACL (Network Access Control List)**: Stateless firewall at the subnet level.

### 4. Connectivity
- **VPC Peering**: Connecting two VPCs.
- **VPC Endpoints**: Private connection to supported AWS services (S3, DynamoDB).
    - Gateway Endpoint
    - Interface Endpoint (PrivateLink)
